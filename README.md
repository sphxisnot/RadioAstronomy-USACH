# RadioAstronomy-USACH
Desarrollo de software y otro código para el procesamiento de datos de Radioastronomía Amateur.

## Créditos
[Observatorio amateur HawkRAO](https://sites.google.com/view/hawkrao/home), en particular a sus notas técnicas de sus observaciones del púlsar Vela, que fueron esenciales para implementar la conversión de IQ a filterbank.


[bin2fil-py](https://github.com/gio54321/bin2fil-py), en cuyo código nos basamos para la construcción del _header_.
